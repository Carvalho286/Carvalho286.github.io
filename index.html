Visualização interativa do design da aplicação

Como visualizar e navegar:
  • Use o scroll do rato para mover verticalmente.
  • Clique e arraste para mover lateralmente.
  • Use a roda do rato para fazer zoom in/out.
  • Em dispositivos táteis, use dois dedos para fazer zoom.
  • Pode colocar em ecrã inteiro usando o botão do Figma (canto superior direito).
Para melhor experiência, recomenda-se visualizar num computador ou tablet.
